RubyCocoa
This application was created using the RubyCocoa bridge. RubyCocoa is available from http://rubycocoa.sourceforge.net/HomePage.

Flickr API Kit
rflickr library from http://rubyforge.org/projects/rflickr/. Some minor modifications made to accommodate the latest flickr API changes. Full library is included in the distribution to eliminate dependency on ruby-gems.

Icon
Web Miner icon from http://www.pixelpressicons.com/

Logo
flickrPreviewR logo was created on http://flickr.nosv.org/

Motivation
Thanks to the people at Flock for creating the latest release version of their browser. It is so buggy and such a memory hog that I am forced to create this application to view my favorite Flickr photos without all the bloat.
